## Requirements

- Java 11

## Install

```sh
mvn clean install
```

## Usage

```sh
cd pairone-rest
mvn spring-boot:run
```

## Deploy

```sh
mvn clean package
cd pairone-rest
mvn appengine:deploy -Dapp.deploy.projectId=pairone
```
